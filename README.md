# Full-stack learning project

**Backend**

- .NET
- Microservices
- Wolverine
- .NET Aspire
- YARP
- Caching
- JWT authentication (HTTP-only cookies)
- Image upload
- Integration tests

**Frontend**

- Angular
- Angular CDK
- NgRx SignalStore
- Tests using Vitest

# How to run

Prerequisites: .NET 10, pnpm, Docker.

`dotnet run --project .\Petrichor.AppHost\`
